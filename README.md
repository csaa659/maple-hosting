## Unmetered Bandwidth: The Option for High-Traffic Use Cases

Standard plans include 50–100 TB of metered monthly bandwidth, which is plenty for most websites and applications. But if you're running video streaming, a large CDN node, high-traffic download servers, or anything where bandwidth predictability matters, the **unmetered bandwidth upgrade** turns any server into a true no-cap machine.

All servers in the lineup can be configured with an unmetered 1 Gbps port (included in base pricing for most models on the unmetered plan). For heavier workloads, Maple-Hosting also offers 5 Gbps, 10 Gbps, and 20 Gbps unmetered port options — which is legitimately impressive for a boutique provider.

This is one area where Maple-Hosting genuinely stands out. Getting 10 Gbps unmetered bandwidth on a dedicated server — with a quality network behind it — isn't something every host offers, and certainly not with the reliability track record these guys have.
